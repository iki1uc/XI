# XI · NC-Station · Kernmodul

XI ist die zentrale NC-Station des iki1uc-Systems.
Sie verbindet Scan, Kernel, Vektor und Matrix zu einem vollständigen NC-Kern.

──────────────────────────────────────────────
## Achsen

- scan – Erkennungsachse
- kernel – Kernoperator
- vector – Vektorachse
- matrix – Matrixachse

──────────────────────────────────────────────
